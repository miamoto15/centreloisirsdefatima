## Notre engagement

Le Centre des Loisirs de Fatima, Î.M., INC. s'engage à rendre son site web accessible au plus grand nombre de personnes possible, incluant les personnes en situation de handicap. Nous nous efforçons continuellement d'améliorer l'accessibilité et l'expérience utilisateur pour tous.

## Standards d'accessibilité

Nous visons à respecter les **Règles pour l'accessibilité des contenus Web (WCAG) 2.1** au niveau AA, une norme internationalement reconnue. Ces lignes directrices expliquent comment rendre les contenus web plus accessibles aux personnes handicapées, notamment celles ayant :

- Une déficience visuelle
- Une déficience auditive
- Des limitations motrices
- Des déficiences cognitives

## Mesures d'accessibilité

Notre site intègre plusieurs fonctionnalités pour améliorer l'accessibilité :

### Navigation

- **Structure claire et cohérente** : Navigation organisée de façon logique
- **Lien "Aller au contenu"** : Permet de passer directement au contenu principal
- **Navigation au clavier** : Toutes les fonctionnalités sont accessibles au clavier
- **Indicateurs de focus** : Les éléments actifs sont clairement identifiés

### Contenu

- **Titres descriptifs** : Hiérarchie logique des titres pour faciliter la navigation
- **Textes alternatifs** : Descriptions pour les images informatives
- **Liens explicites** : Les textes des liens décrivent clairement leur destination
- **Contraste suffisant** : Texte et arrière-plans offrent un bon contraste

### Technologie

- **HTML sémantique** : Utilisation de balises appropriées pour la structure
- **Compatibilité avec les lecteurs d'écran** : Optimisé pour les technologies d'assistance
- **Design adaptatif** : Site utilisable sur différents appareils et tailles d'écran
- **Pas de cookies de suivi** : Respect de votre vie privée

## Limitations connues

Malgré nos efforts, certains aspects du site peuvent présenter des défis d'accessibilité :

- Certaines images anciennes peuvent manquer de descriptions alternatives complètes
- Certains contenus externes ou intégrés peuvent ne pas être entièrement accessibles
- Nous travaillons continuellement à identifier et corriger ces limitations

## Technologies utilisées

Ce site fonctionne grâce à :

- HTML5
- CSS3
- JavaScript minimal pour les interactions essentielles
- Hugo (générateur de site statique)

Le site est conçu pour fonctionner avec :

- Les lecteurs d'écran récents (JAWS, NVDA, VoiceOver)
- Les navigateurs modernes (Chrome, Firefox, Safari, Edge)
- Les outils d'agrandissement d'écran

## Retour d'information et assistance

Nous accueillons favorablement vos commentaires sur l'accessibilité de notre site. Si vous rencontrez des difficultés ou des obstacles à l'accessibilité, veuillez nous contacter par courriel à [info@centreloisirsfatima.com](mailto:info@centreloisirsfatima.com).

Nous nous efforcerons de vous fournir l'information dans un format accessible ou de vous proposer une alternative appropriée.

## Évaluation et amélioration continue

Nous nous engageons à :

- Évaluer régulièrement l'accessibilité de notre site
- Former notre personnel aux meilleures pratiques d'accessibilité web
- Intégrer l'accessibilité dès la conception de nouvelles fonctionnalités
- Tenir compte des retours des utilisateurs pour améliorer l'expérience
